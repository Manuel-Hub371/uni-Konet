# uniKonet Admin Dashboard

This is the administration dashboard for the uniKonet project, built with React and Material UI.

## Prerequisites

- Node.js installed

## Setup

1.  Navigate to the `admin-dashboard` directory.
2.  Install dependencies (once `package.json` is initialized):
    ```bash
    npm install
    ```
    *Required dependencies:* `react`, `react-dom`, `react-router-dom`, `@mui/material`, `@emotion/react`, `@emotion/styled`.

## Running the App

- Development mode:
  ```bash
  npm start
  ```
- Build for production:
  ```bash
  npm run build
  ```

## Folder Structure

-   `public/`: Static assets.
-   `src/components/`: Reusable UI components.
-   `src/pages/`: Application pages (views).
-   `src/services/`: API service calls.
-   `src/utils/`: Helper functions.
